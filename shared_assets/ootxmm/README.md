# OoTxMM Shared Asset PackThis folder is the staging area for cross-game item models.The randomizer writes stable model ids such as `xmm.mask.couple`, `xoot.rocs_feather`, and `xshared.heart_piece` into `crossover_placements.json`. Both ports should treat those ids as the contract between item placement and rendering. The actual asset files can then be packed into an `.o2r`/`.otr` mod archive for each game's `mods` folder without changing the seed format.Current phases:1. Assign stable model ids in the generated placement manifest.2. Port native item object/display-list resources into this shared pack.3. Add draw adapters in SoH and 2Ship that resolve a shared model id to a local or packed display list.4. Replace each placeholder item family with the real model, starting with shared items, masks, hookshots, songs, bottles, and unique major items.The `manifest.ootxmm-assets.json` file intentionally allows `status: "pending-port"` entries. That lets the randomizer know the desired model identity before every display list has been packed and scaled.
